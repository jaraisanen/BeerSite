## A site for a fictional brewery
### Core dependencies and styling:
* React Router
* Material UI
* SaSS

### Required to run:
* Node.js
* NPM / Yarn

### To get started
```
git clone git@github.com:jaraisanen/BeerSite.git
```
Install node_modules 
```
npm install
```
Start the app
```
npm start
```
Open the app on http://localhost:8888
